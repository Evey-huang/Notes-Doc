# **浏览器标准模式、几乎标准模式和怪异模式之间的区别是什么？**

## 三种模式之间的区别

- 标准模式(standards mode)：页面按照HTML与CSS的定义渲染
- 怪异模式(quirks mode)：会模拟更旧的浏览器行为
- 近乎标准模式(almost standards mode)：会实施一种表单元格尺寸的怪异行为(与IE7之前的单元格布局方式一致)，除此之外符合标准定义（`目前该模式基本淘汰了`）
